# CV Diego Pérez Pirela — Sitio web

Currículum en HTML responsive listo para publicarse en **GitHub Pages** y compartirse por **código QR**.

## 1. Editar
Abre `index.html` y busca los comentarios `<!-- ... -->`. Para agregar una experiencia nueva,
copia un bloque `<div class="item"> ... </div>` dentro de la sección **Experiencia** y cámbiale el texto.

## 2. Publicar en GitHub Pages
1. Crea un repositorio público, p. ej. `cv` o `diegoperez`.
2. Sube `index.html` (y este README).
3. En GitHub: **Settings → Pages → Branch: `main` / root → Save**.
4. Tu CV queda en: `https://TU-USUARIO.github.io/cv/`

> Truco: si nombras el repo `TU-USUARIO.github.io`, la URL será directamente `https://TU-USUARIO.github.io/`.

## 3. Generar el código QR
Usa la URL final en cualquier generador (qr-code-generator.com, qrcode-monkey.com).
Descarga el QR en **SVG o PNG de alta resolución** para que se vea nítido impreso/en pantalla.

## 4. Apple Wallet
Crea el pase en una app como **WalletPasses** o **Pass2U**: pega la URL del CV como el
código QR del pase. Al mostrarlo, la persona escanea y abre tu CV sin descargar nada.
